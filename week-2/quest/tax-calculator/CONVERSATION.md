# 세금 계산기 (Tax Calculator) - 대화 내용 정리

## 1차: 앱 생성

**요청:** 종합소득세와 부가가치세 중 원하는 세금 종류를 선택해서 계산할 수 있는 세금 계산기 앱 생성. 조건 분기로 세금 종류에 따라 계산 방식이 달라지도록 구현.

**구현 내용:**
- React + Tailwind CSS CDN 기반 단일 HTML 파일
- 탭으로 세금 종류 선택 (종합소득세 / 부가가치세)
- **종합소득세**: 8단계 누진세율 구간별 계산 (6%~45%)
  - `INCOME_TAX_BRACKETS` 배열에 구간 정의
  - `for` 반복문으로 구간 순회하며 세금 누적 계산
  - 적용 구간, 실효세율, 구간별 비중 바 차트 표시
- **부가가치세**: 10% 세율 계산
  - `if/else` 조건 분기로 세금 포함/미포함 선택에 따라 다른 공식 적용
  - 공급가액, 부가세, 합계 + 단계별 계산 과정 표시
- 빠른 금액 입력 버튼 (+100만, +500만, +1,000만, +1억)
- 한국 원화 형식 콤마 포맷팅

## 2차: 세무 용어 쉽게 수정 + 입력 구조 변경

**요청:** '과세표준' 같은 어려운 세무 용어를 사용자가 이해하기 쉬운 표현으로 변경. 종합소득세는 '1년 동안 번 돈', '사업에 쓴 비용', '공제 금액'으로 입력받고, 내부적으로 과세표준을 계산하도록 구현.

**수정 내용:**

### 입력 구조 변경 (종합소득세)
- 기존: 단일 금액 입력 (과세표준 직접 입력)
- 변경: 3개 필드로 분리
  - "1년 동안 번 돈" (월급, 사업 수입 등을 모두 합한 금액)
  - "사업에 쓴 비용" (재료비, 임대료 등)
  - "공제 금액" (인적공제, 보험료 등 빼주는 금액)
- 내부 계산: `세금 매기는 기준 금액 = 번 돈 - 비용 - 공제`
- 파란색 안내 박스로 기준 금액 계산 과정 실시간 표시

### 용어 변경

| 기존 (세무 용어) | 변경 후 (쉬운 표현) |
|---|---|
| 과세표준 (소득금액) | 세금 매기는 기준 금액 |
| 산출세액 | 내야 할 세금 |
| 실효세율 | 실제 세금 비율 |
| 세후 소득 | 세금 내고 남는 돈 |
| 공급가액 | 물건값 (세금 빼고) |
| 부가가치세 (10%) | 붙는 세금 (10%) |
| 합계 금액 | 총 내야 할 금액 |
| 세금 미포함 금액 | 세금 빼고 적은 금액 |
| 적용 세율 구간 | 어떤 세금 구간이 적용됐나요? |
| 구간별 계산 내역 | 구간별로 세금이 얼마씩 나왔나요? |
| 계산 과정 | 이렇게 계산했어요 |

- 부가가치세 토글에 보조 설명 추가 ("물건값만 입력하면, 세금(10%)을 더해서 총 얼마인지 알려줘요" 등)
- 하단 안내 문구도 쉬운 말투로 변경

## 최종 기술 스택

- React 18 (CDN)
- Tailwind CSS (CDN)
- Babel Standalone (JSX 변환)
