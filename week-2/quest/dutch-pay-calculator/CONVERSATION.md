# 더치페이 계산기 (Dutch Pay Calculator) - 대화 내용 정리

## 1차: 앱 생성

**요청:** 인원수, 각자 먹은 메뉴, 할인 적용이 가능한 더치페이 계산기 앱 생성. 배열로 사람별 주문 메뉴와 가격을 관리하고, 반복문으로 총금액과 개인별 금액을 계산. 조건 분기로 할인 여부와 할인 방식(정액/정률)에 따라 최종 금액이 다르게 나오도록 구현.

**구현 내용:**
- React + Tailwind CSS CDN 기반 단일 HTML 파일
- 사람 추가/삭제 (이름 입력)
- 사람별 주문 메뉴 배열 관리 (`person.orders[]`)
- `for` 반복문으로 총금액, 개인별 소계 계산
- `if/else if/else` 조건 분기로 3가지 할인 모드 처리:
  - 할인 없음
  - 정액 할인 (고정 금액 차감)
  - 정률 할인 (퍼센트 차감)
- 할인은 각자 소비 비율에 따라 비례 분배
- 반올림 오차는 마지막 사람에게 조정
- 사람별 최종 결제 금액 카드 형태로 출력

## 2차: 용어 쉽게 수정

**요청:** 옵션 용어를 초등학생도 이해할 수 있는 쉬운 언어로 변경. 각 버튼에 짧은 설명 문구(보조 텍스트) 추가.

**수정 내용:**

| 기존 용어 | 변경 후 |
|---|---|
| 할인 설정 | 할인 선택 |
| 할인 없음 | 할인 안 하기 |
| 정액 할인 | 금액으로 깎기 |
| 정률 할인 | 퍼센트로 깎기 |
| 할인 금액 (원) | 깎을 금액 (원) |
| 할인율 (%) | 깎을 퍼센트 (%) |
| 할인 적용 | 할인으로 깎인 금액 |

- 각 할인 버튼에 보조 설명 추가:
  - "할인 안 하기" → "원래 가격 그대로 내요"
  - "금액으로 깎기" → "예: 5,000원 깎기"
  - "퍼센트로 깎기" → "예: 10% 깎기"
- 안내 문구도 쉬운 표현으로 변경 ("전체 금액에서 여기에 쓴 만큼 빼줘요" 등)

## 최종 기술 스택

- React 18 (CDN)
- Tailwind CSS (CDN)
- Babel Standalone (JSX 변환)
- Google Fonts (Noto Sans KR)
