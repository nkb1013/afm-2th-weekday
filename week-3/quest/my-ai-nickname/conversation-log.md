# AI 별명 생성기 - 대화기록

## 1차: 프로젝트 생성 요청

**사용자:**
week-3/quest/my-ai-nickname 여기에 AI 별명 생성기를 만들어줘.
사용자가 이름, 성격, 취미 등을 입력하면 AI가 재미있는 별명을 여러 개 생성해주고 마음에 드는 별명을 선택/복사할 수 있게 해줘.

**Claude:**
기존 프로젝트(my-ai-chat) 패턴을 참고하여 `server.js`와 `index.html` 두 파일을 생성했습니다.

- **server.js** — OpenAI API(gpt-4o-mini)를 사용해 별명을 생성하는 Node.js 서버 (포트 3003)
  - `POST /api/nickname` 엔드포인트: 이름, 성격, 취미를 받아 AI에게 별명 5개를 JSON으로 생성 요청
  - 시스템 프롬프트로 JSON 형식 응답을 강제
  - 정적 파일(index.html) 서빙 포함

- **index.html** — 프론트엔드 UI
  - 이름(필수), 성격, 취미 입력 폼
  - 로딩 스피너, 에러 표시
  - 별명 카드 형태로 결과 표시 (클릭 시 클립보드 복사)
  - "다른 별명 다시 만들기" / "처음부터 다시" 버튼
  - 보라색 그라데이션 테마의 깔끔한 디자인

서버 실행 시 포트 3002가 이미 사용 중이어서 3003으로 변경 후 정상 실행 확인.

## 2차: 스타일 옵션 추가 요청

**사용자:**
'할리우드 스타일로' 'k-drama 스타일로' '귀여운 동물 스타일로' 뭐 이런 옵션을 고를 수 있게 기능 추가해줘

**Claude:**
서버와 프론트엔드 모두 수정하여 6가지 스타일 선택 기능을 추가했습니다.

### server.js 변경사항:
- `STYLE_PROMPTS` 객체 추가 — 6가지 스타일별 프롬프트 정의
  - `hollywood`: 할리우드/서양 스타일
  - `kdrama`: K-Drama 스타일
  - `animal`: 귀여운 동물 스타일
  - `game`: 게임/판타지 스타일
  - `food`: 음식 스타일
  - `random`: 자유 스타일 (기본값)
- API에서 `style` 파라미터를 추가로 받아 해당 스타일 프롬프트를 사용자 메시지에 포함
- 시스템 프롬프트에서 "사용자가 요청한 스타일에 맞춰" 문구 추가

### index.html 변경사항:
- 3x2 그리드의 라디오 버튼 카드 UI 추가 (취미 입력 아래, 제출 버튼 위)
  - 🎲 랜덤 (기본 선택)
  - 🎬 할리우드
  - 📺 K-Drama
  - 🐾 귀여운 동물
  - 🎮 게임/판타지
  - 🍕 음식
- CSS: `.style-grid`, `.style-option` 스타일 추가 (선택 시 보라색 강조)
- JS: `style` 값을 읽어 API 요청 body에 포함하도록 수정

서버 재시작 후 정상 동작 확인.
