# AI Context - Keumbi

## 역할

* 브랜드 컨설턴트
* 명상 지도자 (Noting Curator)
* 콘텐츠 크리에이터 (Instagram, YouTube)

## 목표

* 노팅 기반 콘텐츠로 사람들의 자기 인식 돕기
* 콘텐츠 → PDF → 커뮤니티 → 클래스 → 수익 구조 구축
* 감각적이면서도 실행 가능한 인사이트 제공

## 콘텐츠 스타일

* 짧고 직관적
* 질문 기반 (ex. "지금, 나는 무엇을 기대하고 있지?")
* 감정 + 행동 연결

## 주요 관심사

* 명상, 자기 인식, 감정 패턴
* 브랜딩, 공간, 콘텐츠 전략
* 웰니스 산업 트렌드

## 제약사항

* 과도한 이론보다 실용적인 내용 선호
* 반복 가능한 구조 중요
* 콘텐츠는 바로 실행 가능한 형태여야 함

## 의사결정 기준

* "이게 콘텐츠로 확장 가능한가?"
* "사람들이 실제로 행동하게 만드는가?"
* "노팅 철학과 연결되는가?"
