## 1. 프로젝트 개요

**프로젝트 이름**: 노팅힐 (Noting Heal)
**한 줄 정의**: 이 앱은 자기이해 서비스가 아니라, 자기인식 도구다.
**Tagline**: See the frame, not the answer. / "생각이 멈추지 않을 때."

**핵심 동기**: 일상에서 알아차림을 쉽게 하는 방법을 전하는 것

---

## 2. 기획 발전 과정 (대화 흐름)

### Phase 1: 초기 MISSION.md (기존)
- 1회성 AI 비추기 도구
- 회원가입 없음, 기록 안 남김
- AI가 사용자의 상황을 판단 없이 비춰주고 끝
- 타겟: 30~40대 여성

### Phase 2: 습관 리추얼 앱으로 전환
**결정**: 1회성 → Daily Ritual 습관 앱으로 전향
- 로그인 추가 (기록 보존 + 커뮤니티 확장 대비)
- v1 핵심: 감정 선택 + 한 문장 노팅 + 기록 열람
- 타겟 확장: 30~40대 여성 → 20~40대
- 성공 지표: 리텐션 기반 (D1 30%, D7 15%)

### Phase 3: 패턴 시각화 추가
**문제 인식**: "감정 기록 앱"으로만 느껴짐, 경쟁력 부족
**해결**: 패턴 시각화를 v1 핵심 기능으로 추가
- 주간 감정 통계
- 자주 등장한 단어/키워드
- 감정 흐름 캘린더
- "AI가 해석하는 게 아니라, 데이터가 비춰준다" = 핵심 차별점

### Phase 4: 세계관 구축 - 파란 대문
**아이디어**: 노팅힐 영화의 파란 대문을 앱의 시각적 아이덴티티로
- 앱을 열면 파란 대문 → 문 열면 서비스 시작
- "문을 연다" = 일상과 알아차림의 경계
- 바이럴 요소: 스크린샷 한 장으로 전달 가능

### Phase 5: Self Check-in + 맞춤 노팅방법 추천 (통합 플로우)
**결정**: 기존 감정선택→노팅 플로우를 대체하는 통합 플로우
1. 파란 대문 열기
2. Self Check-in (공간/감정/시간)
3. 맞춤 노팅방법 추천 (조합에 따라 다른 방법)
4. 수행
5. 한 문장 기록
6. 패턴 축적

**철학 정리**: "답을 주지 않는다" + "도구를 건넨다"
- "불안하니까 이걸 해봐" (X) — 감정 진단/처방
- "오피스에서 1분 안에 해볼 수 있는 방법이에요" (O) — 상황 기반 도구 제안

### Phase 6: 노팅 룸 + 호텔 세계관
**아이디어**: 노팅힐을 하나의 호텔로. 3개 층, 각 층에 방.

| 층 | 이름 | 넘버링 | 예시 |
|---|---|---|---|
| 1층 | 감정 룸 | Room 1xx | Room 107 - SNS 보다 내가 작아질 때 |
| 2층 | 공간 룸 | Room 2xx | Room 201 - 지하철에서 눈을 감았을 때 |
| 3층 | 시간 룸 | Room 3xx | Room 303 - 잠들기 전, 머릿속이 멈추지 않을 때 |

- 3개 층 = 앱 Self Check-in 3축 (감정/공간/시간) 1:1 매핑
- 넘버링이 곧 내비게이션 (번호만 봐도 어떤 층인지 알 수 있음)

### Phase 7: 인스타-앱 세계관 통일
**인스타 = 노팅힐의 로비**

3가지 콘텐츠 포맷:
1. **노팅 룸** (메인 카드뉴스): 한 회 = 한 방. 카드 8장. 콘텐츠 자산 축적.
2. **오늘의 체크인** (데일리 단일 카드): 앱 Self Check-in 예고. 댓글 후크 최강.
3. **이번 주 노팅 메뉴** (위클리 큐레이션): 추천 룸 5개. 신규 팔로워 진입로.

시각 모티브: 카드뉴스 표지마다 파란 대문 + 룸 넘버 황동키택

### Phase 8: 브랜드 톤 전환
기존: "언젠간 성공할 거라 믿는 사람은 오지 마세요"
변경: **공간 메타포 기반 카피**
- 인사: "환영해요" / "오늘도 들어와 주셨네요"
- 마무리: "잘 머물다 가세요" / "다음 방에서 만나요"
- 댓글 후크: "오늘은 어떤 방이 필요한가요?"

### Phase 9: 최종 고객 여정 확정
건물 발견 → 문 두드리기 → 셀프 체크인 → 방 배정 → 노팅 수행 → 한 줄 쓰기 → 체크아웃

### Phase 10: 비주얼 디자인 확정
- hand-drawn 건축 입면도 스타일
- 크림/베이지 종이 배경 (#EDE9E3) + 딥 네이비 잉크 (#1a1a5e)
- 실제 일러스트 이미지 제작 (입면도 + 대문 클로즈업)
- 방문한 방만 navy shade 처리
- 모든 페이지 톤앤매너 통일

## 3. 최종 확정 사항

### 핵심 철학
- 답을 주지 않는다
- 해석하지 않는다
- 라벨링하지 않는다
- 도구를 건넨다
- "지금 내 안에서 뭐가 작동하고 있는지"를 비추는 도구

### v1 핵심 기능
1. **건물 입면 (메인 홈)** -- 3층 호텔 구조, 방문한 방 shade 표시
2. **파란 대문 (진입)** -- 세계관 입구
3. **Self Check-in** -- 공간/감정/시간 3가지 선택
4. **맞춤 노팅방법 추천** -- 조합에 따른 도구 제안
5. **한 문장 노팅** -- 수행 후 기록
6. **기록 열람 + 패턴 시각화** -- 감정 통계, 반복 단어, 캘린더

### Anti-Scope (v1에서 안 하는 것)
- 커뮤니티/소셜 기능
- AI 해석/분석/조언/라벨링/진단
- Weekly/Monthly 리추얼
- eat/move/rest 모듈

### 기술 스택
- Next.js (App Router)
- Supabase (Auth + PostgreSQL + RLS)
- Tailwind CSS
- Vercel 배포

### 성공 지표
- 첫 리추얼 완료율: 70% 이상
- D1 리텐션: 30% 이상
- D7 리텐션: 15% 이상
- 진짜 성공: "다르게 보이기 시작했다" 피드백 1건

### 타겟 유저
20~40대. 자기 인식 욕구, 번아웃 경험, 기존 자기계발 피로감.
직장인, 창작자, 감정 기복 있는 사람, 명상 관심 있지만 지속 못한 사람.
